PIPELINE OVERVIEW:
================================================================================
- 8 pipeline scripts (A-H execution order + Z orchestrator)
- 7 configuration files (01-07 numbered by usage)
- 8+ stage output files (generated during execution)
- Pipeline processes Google AI models through modalities and deployment

PIPELINE SCRIPTS (in 01_scripts/):
================================================================================

A_fetch_api_models.py                      - Fetches Google AI model data from API

B_filter_models.py                         - Filters models by criteria

C_scrape_modalities.py                     - Scrapes model modalities from web sources

D_enrich_modalities.py                     - Enriches modalities with additional data

E_create_db_data.py                        - Creates database-ready data

F_compare_pipeline_with_supabase.py        - Compares pipeline with Supabase data

G_refresh_supabase_working_version.py      - Updates Supabase working version

H_deploy_to_supabase_ai_models_main.py     - Deploys to main AI models table

Z_run_A_to_F.py                            - Interactive pipeline orchestrator (A-F)

INTERACTIVE ORCHESTRATOR:
================================================================================
Z_run_A_to_F.py offers three execution modes:

1. Run all scripts (A to F)

2. Run script range (e.g., B to E)

3. Run specific scripts (e.g., E, F)
